# Cookie Extractor - Chrome Extension

A Chrome extension that displays website cookies and copies them to your clipboard with one click.

## What It Does

- Reads cookies from the current website
- Displays all cookies in a popup
- Click any cookie to copy its value to clipboard
- Works entirely locally - no data collection or external requests

## Installation (Without Chrome Web Store)

### Quick Install

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **"Developer mode"** (toggle in top-right)
4. Click **"Load unpacked"**
5. Select the `copy-cookie-extension` folder
6. Done! Click the extension icon to use it

### Command Line

```bash
git clone https://github.com/mstoffel-sag/copy-cookie-extension.git
# Then follow steps 2-6 above
```
